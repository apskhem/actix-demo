# Rust Actix-Postgres Demo Implementation
A simple Rust-powered server and PostgreSQL demo application; launched on Heroku

## Implemented Features
- Sending HTML
- Sending JSON
- Receiving JSON
- Variable Route
- Static Files
- Deploying on Heroku
- HTML with Images, CSS, and JS
